# 💳 Credit Card Financial Dashboard

## 📌 Project Overview
The Credit Card Financial Dashboard is an end-to-end Business Intelligence project built using SQL and Power BI. The dashboard provides insights into credit card transactions, customer spending behavior, revenue trends, and key financial KPIs to support data-driven business decisions.

---

## 🎯 Objectives
- Analyze credit card transaction data.
- Monitor revenue and transaction trends.
- Identify top-performing customer segments.
- Track customer spending behavior.
- Create interactive dashboards for business insights.

---

## 🛠️ Tools & Technologies
- Power BI
- SQL (MySQL)
- Power Query
- DAX
- Microsoft Excel

---

## 📊 Dashboard Features
- Revenue Analysis
- Transaction Volume
- Customer Segmentation
- Card Category Analysis
- Weekly & Monthly Trends
- Interest Earned Analysis
- Top States & Cities
- Income Group Analysis
- KPI Cards
- Interactive Filters & Slicers

---

## 📈 Key Insights
- Identified customer segments contributing the highest revenue.
- Analyzed spending patterns across different card categories.
- Compared revenue trends over time.
- Monitored transaction count and customer activity.
- Created interactive dashboards for easy business reporting.
